# Бот вечернего отчёта для Telegram

Бот автоматически формирует вечерний отчёт из файла Excel с экспортом чеков. Отправьте боту файл — получите готовый отчёт.

## Установка

1. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

2. Создайте бота в Telegram: напишите [@BotFather](https://t.me/BotFather), команду `/newbot`, следуйте инструкциям.

3. Скопируйте `config.example.py` в `config.py` и вставьте полученный токен:
   ```
   BOT_TOKEN = "ваш_токен_от_BotFather"
   ```

## Запуск

```
python bot.py
```

## Использование

1. Найдите бота в Telegram
2. Отправьте команду `/start`
3. Отправьте файл Excel (.xlsx) с экспортом чеков
4. Бот вернёт готовый отчёт

Дата в отчёте берётся из названия файла (например, `Экспорт чеков от 17-01-2026.xlsx` → 17.01.2026). Если дату определить не удаётся, используется сегодняшняя.

## Структура Excel

Бот ожидает файл с экспортом чеков, где:
- **Столбец I** — тип операции (Приход / Возврат прихода)
- **Столбец O** — список позиций чека
- **Столбец P** — итог по чеку
- **Столбец Q** — способ оплаты (Безналичными / Наличными)

## Скрипты Windows

- `openbot.bat` — запуск бота  
- `close.bat` — остановка процесса `bot.py`  
- `restart.bat` — перезапуск

## Настройка правил отчёта

Логика категорий и сумм задаётся в `report_parser.py` (функция `parse_excel_report` и вспомогательные функции).

## GitHub

После клонирования создайте `config.py` из `config.example.py` — файл `config.py` в репозиторий не попадает (токен в `.gitignore`).
